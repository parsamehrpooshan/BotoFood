## 🛠 روش نصب و اجرای پروژه

### 📥 نصب

1. کلون کردن ریپازیتوری:

```bash
git clone https://github.com/parsamehrpooshan/Torino-api.git
```

2. ورود به پوشه پروژه:


```bash
cd project-name
```

3. ران کردن سرور | در ترمینال اول :


```bash
npx json-server --watch db.json --port 4000

```

5. اجرای پروژه در حالت توسعه | در ترمینال دوم:


```bash
npm run dev
```

📍 سپس پروژه روی آدرس زیر در دسترس خواهد بود:

```
http://localhost:3000
```
